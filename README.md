# 🤖 kerogrammer.github.io

IT 전문가이자 엔지니어인 **케로그래머(kerogrammer)**의 기술 및 일상 기록 공간입니다. 5년 만의 새로운 시작으로 티스토리에서 깃허브 블로그로 거처를 옮겨 기록을 이어갑니다.

이 블로그는 단순한 정보 나열을 넘어, 실무에서 겪는 고민과 해결 과정, 그리고 AI와 협업하는 미래 지향적인 엔지니어링 일상을 담고 있습니다.

## 🚀 주요 콘텐츠 (Focus Areas)
- **전문 기술 분석:** IT 인프라 설계, 정보 보안 거버넌스, 최신 IT 트렌드 심층 분석.
- **실무 트러블슈팅:** 현장에서 마주치는 복잡한 기술적 문제들에 대한 명쾌한 해결책 기록.
- **AI 협업 실험:** AI 비서 '태비(Tabi)'와 함께 블로그를 자동화하고 관리하는 혁신적인 여정 공유.
- **보드게임 전략:** '여명의 제국 4(TI4)', '듄 임페리움' 등 전략 보드게임에 대한 심도 있는 분석 및 후기.

## 🛠 기술 스택 (Tech Stack)
- **Framework:** [Jekyll](https://jekyllrb.com/)
- **Theme:** [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy/) (Customized)
- **Hosting:** [GitHub Pages](https://pages.github.com/)
- **Analytics:** Google Analytics 4 (GA4)
- **Search:** Brave Search API Integration

## 🤖 AI 비서 '태비(Tabi)' 소개
이 블로그의 운영에는 전담 AI 사원인 **태비(Tabi)**가 함께합니다. 태비는 포스팅 초안 작성, 컨텍스트 관리, 시스템 최적화 등 블로그 운영 전반에 걸친 자동화 업무를 담당하고 있습니다.
