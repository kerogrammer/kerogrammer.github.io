# AI Posting Guideline

이 문서는 블로그 초안을 AI로 생성하고, 사람이 승인한 뒤 발행하기 위한 운영 기준입니다.

## 1. 운영 원칙

- AI는 초안을 작성한다.
- 사람은 제목, 사실관계, 문체, SEO 품질을 검토한다.
- 승인 전까지는 `_drafts/`에만 존재해야 한다.
- 승인된 글만 `_posts/`로 이동하여 발행한다.

## 2. 초안 작성 기준

모든 초안은 아래 front matter를 포함해야 한다.

```yaml
---
layout: post
title: "제목"
date: 2026-04-04 12:00:00 +0900
categories: [Tech, AI]
tags: [ChatGPT, GitHub, SEO]
description: "검색 결과에 노출될 요약문"
author: kerogrammer
---
```

## 3. SEO 작성 기준

- 제목에 핵심 키워드를 포함한다.
- description은 120~160자 내외로 작성한다.
- 본문은 H2/H3 구조를 사용한다.
- 첫 2문단 안에 핵심 키워드를 자연스럽게 포함한다.
- 내부 링크가 가능하면 기존 글과 연결한다.

## 4. 검토 체크리스트

- 제목이 과장되지 않았는가
- 최신 정보가 필요한 영역은 사실 검증이 되었는가
- description이 검색엔진 친화적으로 작성되었는가
- 카테고리와 태그가 적절한가
- 발행 가치가 있는 글인가

## 5. 발행 정책

- AI가 생성한 글은 자동 공개하지 않는다.
- 검토 완료 후 승인된 글만 발행한다.
- 승인 기록은 PR 또는 작업 이력으로 남긴다.
